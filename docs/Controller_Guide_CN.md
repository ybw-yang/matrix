# 遥控器与控制指南

## 手柄控制说明（推荐）

| 操作 | 控制输入 |
|------|---------|
| 站立 / 坐下 | 按住 **LB** + **Y** |
| 前进 / 后退 / 左移 / 右移 | **左摇杆** (上 / 下 / 左 / 右) |
| 向左 / 向右旋转 | **右摇杆** (左 / 右) |
| 向前跳 (冲刺) | 按住 **RB** + **Y** |
| 原地跳 | 按住 **RB** + **X** |
| 翻筋斗 | 按住 **RB** + **B** |

*注意：推荐使用 Logitech Wireless Gamepad F710。*

## 键盘控制说明

| 操作 | 控制输入 |
|------|---------|
| 站立 | U |
| 坐下 | 空格键 (Space) |
| 前进 / 后退 / 左移 / 右移 | W / S / A / D |
| 向左 / 向右旋转 | Q / E |
| 开始 | Enter |

按 **V** 键在自由相机与机器人视角之间切换。
按住 **鼠标左键** 可临时切换到自由相机模式。
